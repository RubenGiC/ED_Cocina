Receta

Es un TDA que nos permite almacenar tanto los datos de una receta como los valores nutricionales de una receta, es decir, sus calorias, hidratos de carbono, grasas, etc...

Resumen Atributos Receta

  • Nombre del Alimento (string)
  • Calorias (float)
  • Nutrientes:
    • Hidratos de Carbono (float)
    • Grasas (float)
    • Proteinas (float)
    • Fibra (float)
  • Codigo (string)
  • Plato (UNSIGNED INT)
  • Listado de Ingredientes (list< pair <string,unsigned int> >)

Principales metodos y sobrecargas :

  • Tanto gets como sets
  • operator >> ( capaz de leer una receta completo )
  • operator << ( nos permite escribir los valores de una receta al completo )
